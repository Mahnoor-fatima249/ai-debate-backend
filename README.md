# 🎙️ AI Debate Assistant (Backend)

This is the core backend for our AI Debate Partner. It handles audio processing, transcribes speech, and generates logical counter-arguments.

## 🚀 Features
* **Speech-to-Text:** Uses OpenAI's **Whisper** model for accurate transcription.
* **AI Reasoning:** Powered by **Llama-3.1-8b** (via Groq) for fast, professional debate responses.
* **FastAPI:** High-performance web framework for the API endpoints.

## 🛠️ Tech Stack
* Python
* FastAPI
* Whisper (STT)
* Groq Cloud API
